Automated Gym Class Bookings

Built an automated bot that books me into the same gym classes every week as soon as the class opens for booking. I was tired of constantly missing classes espically as when I'd go there were lots of no shows, so I used the Selenium libary in Python to book me into classes. It runs on a raspberry pi with chromium scheduler.
